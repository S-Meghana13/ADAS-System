# 🚗 ADAS (Advanced Driver Assistance System)

A real-time computer vision-based Advanced Driver Assistance System (ADAS) built using Deep Learning and OpenCV to enhance driving safety by detecting vehicles, pedestrians, lanes, and potential collisions.

---

## 📌 Features
- 🚘 Vehicle Detection using YOLOv8  
- 🚶 Pedestrian Detection  
- 🛣️ Lane Detection  
- ⚠️ Collision Warning System  
- 🎥 Real-time video processing using OpenCV  
- ⚡ Optimized for real-time performance  

---

## 🛠️ Tech Stack
- **Programming Language:** Python  
- **Deep Learning:** YOLOv8, CNN  
- **Libraries:** OpenCV, NumPy  
- **Frameworks:** PyTorch / TensorFlow  
- **Tools:** Jupyter Notebook, VS Code
